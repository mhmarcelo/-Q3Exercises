HTML

HyperText Markup Language

HyperText Markup Language is what you use to create and develop web pages. It includes tags, elements, and attributes to build the structure of your website. There is a variety of things you can create in an HTML web page such as lists, forms, tables, etc.

CSS

Cascading Style Sheets

Cascading Style Sheets is used for organizng the content of your HTML and its style or design. It simply gives life to your boring black and white HTML web pages. There are various types of CSS such as internal, external, and inline CSS.

Bootstrap

Bootstrap Version 4

Bootstrap is a Responsive Web Design that molds designs into whatever device is being used. It is easily accessible online and offline. A great feature of it would be that it is easy to use. It is also compatible with all modern browsers.
